###### 生成树
   + 定义：连通图的极小连通子图，它包含图中全部的n个顶点，恰好有n-1条边
   + 最小生成树：所有生成树中，权值最小的一颗，适用于有权的连通图。
      + 使用：
         1. 在n个城市之间铺设光缆 
    + 切分定理：
       1. 切分:把图中的节点分成两部分，称为一个切分
       2. 横切边：如果一个边有两个顶点，分别属于切分的两部分，这个切边称为横切边
       3. 切分定理：给定任意切分，横切边中权值最小的边必然属于最小生成树。
